# 7_control.md

# 피드백 제어와 PID 제어에 대한 조사

## 1. 수행목표

로봇을 제어하는 기초 제어 이론을 학습하고, 피드백 제어와 PID 제어의 개념을 이해한다. 제어 시스템의 기본 구조, 목표값과 오차, 오픈 루프와 클로즈드 루프, PID 제어의 P, I, D 항목의 역할과 수학적 표현, PID 튜닝 방법을 정리한다.

---

## 2. 제어 시스템의 기초

### 2.1 제어 시스템의 개념

제어 시스템(control system)은 어떤 장치나 시스템이 원하는 동작을 하도록 입력을 조절하는 시스템이다. 로봇에서는 모터 속도, 바퀴 위치, 로봇팔 각도, 로봇의 방향, 온도, 거리 등을 원하는 값에 맞추기 위해 제어 시스템을 사용한다.

예를 들어 로봇 운반차가 일정한 속도로 이동해야 한다면, 제어 시스템은 모터에 전달되는 전압이나 PWM 값을 조절하여 실제 속도가 목표 속도에 가까워지도록 만든다.

```text
목표값 → 제어기 → 모터/로봇 → 실제 출력
```

---

### 2.2 제어의 목표

제어의 목표는 실제 출력이 목표값에 최대한 가까워지도록 만드는 것이다.

| 제어 대상 | 제어 목표 |
|---|---|
| 모터 속도 | 원하는 속도로 회전 |
| 로봇 위치 | 목표 위치까지 이동 |
| 로봇팔 각도 | 원하는 관절 각도로 이동 |
| 온도 | 설정 온도 유지 |
| 거리 | 장애물과 일정 거리 유지 |
| 방향 | 원하는 진행 방향 유지 |

좋은 제어 시스템은 정확하고, 안정적이며, 목표값 변화에 빠르게 반응해야 한다. 또한 외부 환경 변화에도 출력이 크게 흔들리지 않아야 한다.

---

## 3. 피드백 제어의 기본 개념

### 3.1 피드백이란?

피드백(feedback)은 시스템의 출력 결과를 다시 입력 쪽으로 되돌려 제어에 사용하는 것을 말한다.

예를 들어 모터 속도를 제어할 때, 단순히 PWM 값만 주는 것이 아니라 엔코더로 실제 속도를 측정하고 목표 속도와 비교하여 다시 PWM 값을 조정하는 방식이 피드백 제어이다.

```text
목표값 → 제어기 → 모터 → 실제값
          ↑              ↓
          ← 센서 측정값 ←
```

피드백 제어를 사용하면 실제 결과를 확인하면서 제어할 수 있으므로 더 정확한 제어가 가능하다.

| 현장 적용 사례     | 목표값          | 측정 센서      | 제어 대상   | 피드백 제어 역할                 |
| ------------ | ------------ | ---------- | ------- | ------------------------- |
| 컨베이어 속도 제어   | 설정 속도        | 엔코더        | 모터      | 물건의 무게가 변해도 일정한 속도를 유지    |
| 로봇팔 위치 제어    | 목표 각도·위치     | 엔코더        | 관절 모터   | 로봇팔이 원하는 위치에 정확히 도달하도록 제어 |
| 자동차 크루즈 컨트롤  | 설정 속도        | 속도 센서      | 엔진·브레이크 | 오르막이나 내리막에서도 일정한 속도를 유지   |
| 에어컨 온도 제어    | 설정 온도        | 온도 센서      | 압축기·팬   | 실내 온도를 설정값에 가깝게 유지        |
| 드론 자세 제어     | 목표 자세·각도     | 자이로·가속도 센서 | 프로펠러 모터 | 바람에 기울어진 드론을 다시 수평으로 복원   |
| 모바일 로봇 직진 제어 | 좌우 바퀴의 목표 속도 | 바퀴 엔코더     | 좌우 모터   | 바퀴 속도 차이를 보정하여 직진         |
| 탱크 수위 제어     | 목표 수위        | 수위 센서      | 펌프·밸브   | 액체의 수위를 일정하게 유지           |
| 산업용 온도 제어    | 목표 온도        | 온도 센서      | 히터      | 공정 온도를 정확하게 유지            |

### 피드백 제어의 기본 흐름

목표값 설정 -> 현재값 측정 -> 오차 계산 -> 제어기에서 출력 조절 -> 정지동작 -> 다시 현재값 측정
---

### 3.2 목표값, 실제값, 오차

피드백 제어에서는 목표값과 실제값을 비교한다.

| 용어 | 의미 | 예시 |
|---|---|---|
| 목표값 | 원하는 값 | 목표 속도 1.0 m/s |
| 실제값 | 센서로 측정한 현재 값 | 실제 속도 0.8 m/s |
| 오차 | 목표값과 실제값의 차이 | 1.0 - 0.8 = 0.2 m/s |

오차는 다음과 같이 표현한다.

```text
오차 = 목표값 - 실제값
```

수식으로는 다음과 같이 쓴다.

```text
e(t) = r(t) - y(t)
```

| 기호 | 의미 |
|---|---|
| e(t) | 시간 t에서의 오차 |
| r(t) | 목표값 |
| y(t) | 실제 출력값 |

오차가 0에 가까울수록 실제값이 목표값을 잘 따라가고 있다는 뜻이다.

r(t): Reference, 즉 목표값
y(t): Output, 즉 실제 출력값
e(t): Error, 즉 목표값과 실제값의 차이

---

## 4. 피드백 제어의 기본 구조

### 4.1 제어 시스템의 구성 요소

피드백 제어 시스템은 일반적으로 다음과 같은 요소로 구성된다.

| 구성 요소 | 역할 |
|---|---|
| 목표값 | 시스템이 도달해야 하는 원하는 값 |
| 비교기 | 목표값과 실제값을 비교하여 오차 계산 |
| 제어기 | 오차를 줄이기 위한 제어 입력 계산 |
| 구동기 | 제어 입력을 물리적 동작으로 변환 |
| 제어 대상 | 실제로 제어되는 장치 |
| 센서 | 출력값을 측정하여 피드백으로 전달 |

로봇 모터 속도 제어에서는 다음과 같이 대응된다.

| 제어 요소 | 로봇 예시 |
|---|---|
| 목표값 | 목표 모터 속도 |
| 비교기 | 목표 속도와 실제 속도 비교 |
| 제어기 | PID 제어기 |
| 구동기 | 모터 드라이버 |
| 제어 대상 | DC 모터와 바퀴 |
| 센서 | 엔코더 |

---

### 4.2 오픈 루프 제어

오픈 루프 제어(open-loop control)는 출력 결과를 다시 확인하지 않고 입력만 주는 제어 방식이다.

```text
입력 → 제어기 → 제어 대상 → 출력
```

예를 들어 모터에 PWM 150을 주면 일정 속도로 돌 것이라고 가정하는 방식이다.

| 장점 | 단점 |
|---|---|
| 구조가 단순하다. | 실제 결과를 확인하지 못한다. |
| 구현이 쉽다. | 외란에 약하다. |
| 센서가 필요 없을 수 있다. | 목표값과 실제값이 달라도 보정하지 못한다. |

---

### 4.3 클로즈드 루프 제어

클로즈드 루프 제어(closed-loop control)는 출력 결과를 센서로 측정하여 다시 제어에 사용하는 방식이다. 피드백 제어는 클로즈드 루프 제어에 해당한다.

```text
목표값 → 비교기 → 제어기 → 구동기 → 제어 대상 → 출력
          ↑                                  ↓
          ←------------- 센서 피드백 ---------
```

| 장점 | 단점 |
|---|---|
| 실제값을 확인하며 제어한다. | 센서가 필요하다. |
| 오차를 줄일 수 있다. | 구조가 복잡하다. |
| 외란에 대응할 수 있다. | 제어기 튜닝이 필요하다. |
| 정확한 제어가 가능하다. | 센서 노이즈 영향을 받을 수 있다. |

---

## 5. PID 제어의 개념

### 5.1 PID 제어란?

PID 제어는 피드백 제어에서 가장 널리 사용되는 제어 방법 중 하나이다. PID는 Proportional, Integral, Derivative의 약자이다.

| 항목 | 이름 | 의미 |
|---|---|---|
| P | Proportional | 비례 제어 |
| I | Integral | 적분 제어 |
| D | Derivative | 미분 제어 |

PID 제어기는 목표값과 실제값의 오차를 이용하여 제어 출력을 계산한다. 현재 오차, 누적 오차, 오차 변화 속도를 함께 고려한다.

```text
PID 제어 출력 = P 항 + I 항 + D 항
```

---

### 5.2 PID 제어의 역할

PID 제어의 목적은 오차를 줄여 실제값이 목표값에 가까워지도록 하는 것이다.

| 역할 | 설명 |
|---|---|
| 목표값 추종 | 실제값이 목표값을 따라가도록 함 |
| 오차 감소 | 목표값과 실제값의 차이를 줄임 |
| 외란 보정 | 하중 변화, 마찰 변화 등을 보정 |
| 안정화 | 출력이 심하게 흔들리지 않도록 조절 |

---

## 6. P, I, D 각 항목의 역할과 수학적 표현

### 6.1 P 제어, 비례 제어

P 제어는 현재 오차에 비례하여 제어 입력을 만든다.

```text
P 항 = Kp × e(t)
```

| 기호 | 의미 |
|---|---|
| Kp | 비례 게인 |
| e(t) | 현재 오차 |

| 기호     | 의미                         |
| ------ | -------------------------- |
| (u(t)) | 제어기에 의해 만들어지는 출력 명령        |
| (K_p)  | 오차에 얼마나 강하게 반응할지 정하는 비례 게인 == 가중치값 |
| (e(t)) | 현재 목표값과 실제값의 차이            |



P 제어는 오차가 크면 큰 제어 입력을 만들고, 오차가 작으면 작은 제어 입력을 만든다.

| 장점 | 단점 |
|---|---|
| 반응이 빠르다. | 오차가 완전히 사라지지 않을 수 있다. |
| 구조가 단순하다. | Kp가 너무 크면 진동이 발생할 수 있다. |

---

### 6.2 I 제어, 적분 제어

I 제어는 오차를 시간에 따라 누적하여 제어 입력에 반영한다.
미적분 강의: https://www.youtube.com/watch?v=NHxr1P5D0Aw
```text
I 항 = Ki × ∫ e(t) dt
```

| 기호 | 의미 |
|---|---|
| Ki | 적분 게인 |
| ∫ e(t) dt | 오차의 누적값 |

I 제어는 작은 오차가 오래 지속될 때 이를 누적해서 보정한다. 따라서 P 제어만 사용할 때 남을 수 있는 정상상태 오차를 줄이는 역할을 한다.

| 장점 | 단점 |
|---|---|
| 정상상태 오차를 줄인다. | 너무 크면 오버슈트가 커질 수 있다. |
| 지속적인 작은 오차를 보정한다. | 적분 포화가 발생할 수 있다. |

적분값이 과도하게 누적되어 제어 출력이 너무 커지는 현상을 적분 와인드업(integral windup)이라고 한다.

---

### 6.3 D 제어, 미분 제어

D 제어는 오차가 얼마나 빠르게 변하는지를 보고 제어 입력을 만든다.
미적분 강의: https://www.youtube.com/watch?v=NHxr1P5D0Aw

```text
D 항 = Kd × de(t) / dt
```

| 기호 | 의미 |
|---|---|
| Kd | 미분 게인 |
| de(t) / dt | 오차의 변화율 |

D 제어는 로봇이 목표값에 너무 빠르게 접근하거나 오차가 갑자기 변할 때 이를 완화하는 역할을 한다. 쉽게 말하면 브레이크처럼 진동과 오버슈트를 줄이는 역할을 한다.

| 장점 | 단점 |
|---|---|
| 오버슈트를 줄일 수 있다. | 센서 노이즈에 민감하다. |
| 진동을 줄이고 안정성을 높인다. | 너무 크면 출력이 불안정할 수 있다. |

오버슈트(Overshoot)는 제어 시스템이 목표값에 도달할 때, 목표값을 한 번 넘어가는 현상이야.

---

### 6.4 PID 전체 수식

PID 제어기의 전체 수식은 다음과 같다.

```text
u(t) = Kp e(t) + Ki ∫ e(t) dt + Kd de(t)/dt

제어출력 =(P) 비례게인X목표값과실제값의 오차 + (I) 적분게인X오차의누적값 
+(D) 미분게인X오차의 변화율 
```

| 기호 | 의미 |
|---|---|
| u(t) | 제어 출력 |
| Kp | 비례 게인 |
| Ki | 적분 게인 |
| Kd | 미분 게인 |
| e(t) | 목표값과 실제값의 오차 |
| ∫ e(t) dt | 오차의 누적값 |
| de(t)/dt | 오차의 변화율 |

---

## 7. 로봇에서 PID 제어 예시

### 7.1 모터 속도 제어 예시

목표 속도가 `100 RPM`이고 엔코더로 측정한 실제 속도가 `80 RPM`이라고 가정한다.

```text
오차 = 목표값 - 실제값
오차 = 100 - 80
오차 = 20 RPM
```

PID 제어기는 이 오차를 줄이기 위해 모터 출력 PWM을 증가시킨다. 시간이 지나 실제 속도가 100 RPM에 가까워지면 오차가 작아지고 제어 출력도 조정된다.

---

### 7.2 라인 트레이싱 로봇 예시

라인 트레이싱 로봇에서는 로봇이 검은 선의 중앙을 따라가야 한다. 목표값은 라인의 중앙이고, 실제값은 센서가 측정한 현재 위치이다.

| 상황 | 제어 동작 |
|---|---|
| 로봇이 선의 왼쪽으로 벗어남 | 오른쪽으로 조향 |
| 로봇이 선의 오른쪽으로 벗어남 | 왼쪽으로 조향 |
| 로봇이 중앙에 있음 | 직진 |

PID 제어를 사용하면 선에서 벗어난 정도와 변화 속도를 반영하여 더 부드럽게 라인을 따라갈 수 있다.

---

## 8. PID 튜닝

### 8.1 PID 튜닝이란?

PID 튜닝은 `Kp`, `Ki`, `Kd` 값을 적절하게 조정하는 과정이다. 같은 PID 제어기라도 게인 값에 따라 로봇의 움직임이 크게 달라진다.

| 게인 | 너무 작을 때 | 너무 클 때 |
|---|---|---|
| Kp | 반응이 느림 | 진동, 오버슈트 발생 |
| Ki | 정상상태 오차가 남음 | 오버슈트, 적분 포화 발생 |
| Kd | 진동 억제 효과 부족 | 노이즈에 민감, 출력 불안정 |

PID 튜닝의 목표는 빠르게 목표값에 도달하면서도 진동과 오버슈트를 줄이는 것이다.

---

### 8.2 기본적인 수동 튜닝 방법

1. `Ki`와 `Kd`를 0으로 둔다.
2. `Kp`를 조금씩 올려 시스템이 목표값에 반응하도록 한다.
3. 출력이 심하게 진동하기 직전까지 `Kp`를 조절한다.
4. `Kd`를 조금씩 올려 진동과 오버슈트를 줄인다.
5. 목표값 근처에 오차가 계속 남으면 `Ki`를 조금씩 올린다.
6. 오버슈트나 진동이 커지면 `Kp`, `Ki`, `Kd`를 다시 낮추며 조정한다.

---

### 8.3 P, I, D 값을 조정할 때의 영향

| 조정 항목 | 증가시키면 나타나는 경향 |
|---|---|
| Kp 증가 | 반응이 빨라지지만 진동과 오버슈트가 증가할 수 있다. |
| Ki 증가 | 정상상태 오차가 줄어들지만 오버슈트와 진동이 증가할 수 있다. |
| Kd 증가 | 오버슈트와 진동을 줄일 수 있지만 노이즈에 민감해질 수 있다. |

PID 튜닝은 한 번에 완벽한 값을 찾기 어렵기 때문에 작은 값부터 천천히 조정하며 실험하는 것이 중요하다.

---

### 8.4 지글러-니콜스 튜닝

지글러-니콜스(Ziegler-Nichols) 방법은 대표적인 PID 튜닝 방법 중 하나이다. 먼저 `Ki`와 `Kd`를 0으로 두고 `Kp`를 증가시켜 시스템이 일정하게 진동하는 지점을 찾는다. 이때의 임계 게인과 진동 주기를 이용하여 `Kp`, `Ki`, `Kd` 값을 정한다.

다만 실제 로봇에서는 이 방법으로 얻은 값이 너무 공격적일 수 있으므로, 안전하게 작은 값부터 실험하며 조정하는 것이 좋다.

## 9. 오픈 루프와 클로즈드 루프 비교 예시

로봇 운반차의 모터 속도 제어를 예로 들면 다음과 같다.

| 구분 | 오픈 루프 제어 | 클로즈드 루프 제어 |
|---|---|---|
| 센서 사용 | 사용하지 않음 | 엔코더 사용 |
| 제어 방식 | PWM 값을 그대로 입력 | 목표 속도와 실제 속도 비교 |
| 외란 대응 | 어려움 | 가능 |
| 정확도 | 낮을 수 있음 | 높음 |
| 예시 | PWM 150을 계속 출력 | 실제 속도를 보고 PWM 조정 |

오픈 루프 제어는 같은 PWM을 계속 출력하지만, 클로즈드 루프 제어는 엔코더로 실제 속도를 확인하고 오차를 보정한다.

## 10. 결론

제어 시스템은 로봇이 원하는 동작을 하도록 입력을 조절하는 시스템이다. 제어의 목표는 실제 출력이 목표값에 최대한 가까워지도록 만드는 것이다.

피드백 제어는 실제 출력값을 센서로 측정하여 다시 제어에 사용하는 방식이다. 목표값과 실제값의 차이를 오차라고 하며, 제어기는 이 오차를 줄이는 방향으로 제어 입력을 계산한다.

PID 제어는 현재 오차를 반영하는 P 제어, 누적 오차를 반영하는 I 제어, 오차 변화율을 반영하는 D 제어로 구성된다. PID 제어 수식은 다음과 같다.

```text
u(t) = Kp e(t) + Ki ∫ e(t) dt + Kd de(t)/dt

제어출력 =(P) 비례게인X목표값과실제값의 오차 + (I) 적분게인X오차의누적값 
+(D) 미분게인X오차의 변화율 
```

PID 튜닝은 `Kp`, `Ki`, `Kd` 값을 조정하는 과정이며, 로봇이 빠르고 안정적으로 목표값을 따라가도록 만드는 데 필요하다.
