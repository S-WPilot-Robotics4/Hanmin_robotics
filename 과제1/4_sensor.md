# 4. Sensor

## 1. 기구로서의 센서

### 1.1 센서의 정의 및 역할

센서란 주변 환경이나 시스템의 물리적, 화학적, 기계적 변화를 감지하여 전기적 신호로 변환하는 장치이다.  
로봇이나 자동화 시스템에서 센서는 사람이 눈, 귀, 피부 등을 통해 외부 정보를 인식하는 것과 비슷한 역할을 한다.

센서의 주요 역할은 다음과 같다.

- 외부 환경의 상태를 감지한다.
- 로봇이나 기계가 현재 상태를 판단할 수 있도록 정보를 제공한다.
- 제어 시스템이 적절한 동작을 수행하도록 입력 데이터를 제공한다.
- 자동화, 안전 제어, 위치 인식, 거리 측정, 라인 감지 등에 사용된다.

예를 들어 라인트레이서 로봇에서는 IR 센서를 이용하여 바닥의 검은색 선과 흰색 바탕을 구분하고, 그 결과를 바탕으로 모터의 방향과 속도를 제어한다.

---

### 1.2 센서의 유형과 용도

센서는 측정 대상과 사용 목적에 따라 여러 종류로 나눌 수 있다.

| 센서 종류 | 측정 대상 | 주요 용도 |
|---|---|---|
| 온도 센서 | 온도 | 실내 온도 측정, 장비 온도 감시 |
| 거리 센서 | 물체와의 거리 | 장애물 감지, 로봇 주행 |
| 초음파 센서 | 초음파 반사 시간 | 거리 측정, 장애물 회피 |
| IR 센서 | 적외선 반사량 | 라인 감지, 근접 감지 |
| 조도 센서 | 빛의 세기 | 자동 조명, 밝기 감지 |
| 압력 센서 | 압력 | 공압 제어, 터치 감지 |
| 가속도 센서 | 가속도 | 자세 측정, 충격 감지 |
| 자이로 센서 | 각속도 | 드론, 로봇 자세 제어 |
| 엔코더 | 회전량, 위치 | 모터 위치 및 속도 측정 |
| 카메라 센서 | 영상 정보 | 객체 인식, 비전 처리 |

센서는 단순히 값을 측정하는 장치가 아니라, 제어 시스템이 판단할 수 있는 데이터를 만들어주는 입력 장치라고 볼 수 있다.

---

### 1.3 센서 신호의 유형: 아날로그 신호와 디지털 신호

센서가 출력하는 신호는 크게 아날로그 신호와 디지털 신호로 나눌 수 있다.

#### 아날로그 신호

아날로그 신호는 연속적인 값을 가지는 신호이다.  
예를 들어 0V에서 5V 사이의 전압이 연속적으로 변하는 형태이다.

예시:

```text
0.5V, 1.2V, 2.8V, 4.7V
```

아날로그 신호는 세밀한 변화량을 표현할 수 있지만, 마이크로컨트롤러에서 사용하려면 ADC(Analog to Digital Converter)를 통해 디지털 값으로 변환해야 한다.

예를 들어 아두이노의 `analogRead()`는 0V에서 5V 사이의 전압을 0부터 1023 사이의 값으로 변환한다.

#### 디지털 신호

디지털 신호는 보통 0 또는 1의 값을 가진다.  
즉, LOW 또는 HIGH 상태로 표현된다.

예시:

```text
0 또는 1
LOW 또는 HIGH
검은색 감지 / 흰색 감지
장애물 있음 / 장애물 없음
```

디지털 신호는 판단이 간단하다는 장점이 있다.  
하지만 아날로그 신호보다 세밀한 변화량을 표현하기는 어렵다.

---

### 1.4 센서의 해상도와 정확도

#### 해상도

센서의 해상도는 센서가 구분할 수 있는 최소 변화량을 의미한다.  
해상도가 높을수록 더 작은 차이를 구분할 수 있다.

예를 들어 10비트 ADC는 입력 전압을 0부터 1023까지 총 1024단계로 나눈다.  
5V 기준이라면 한 단계의 크기는 약 0.00488V이다.

```text
5V / 1024 ≈ 0.00488V
```

즉, 10비트 ADC는 약 4.88mV 단위의 변화를 구분할 수 있다.

#### 정확도

정확도는 센서가 측정한 값이 실제 값과 얼마나 가까운지를 의미한다.  
정확도가 높을수록 실제 값에 가까운 결과를 얻을 수 있다.

예를 들어 실제 온도가 25°C일 때 센서가 24.9°C를 측정하면 정확도가 높은 것이고, 30°C를 측정하면 정확도가 낮은 것이다.

해상도와 정확도는 서로 다른 개념이다.  
해상도가 높다고 해서 반드시 정확도가 높은 것은 아니다.  
센서가 아주 세밀하게 값을 표시하더라도 실제 값과 차이가 크면 정확도는 낮다.

---

### 1.5 응답 시간과 샘플링 속도의 개념

#### 응답 시간

응답 시간은 센서가 환경 변화를 감지하고 출력 신호가 안정적으로 변하기까지 걸리는 시간을 의미한다.  
응답 시간이 짧을수록 빠르게 변화하는 상황에 잘 대응할 수 있다.

예를 들어 라인트레이서 로봇이 빠르게 움직일 때, IR 센서의 응답 시간이 느리면 검은색 선을 늦게 감지하여 경로를 벗어날 수 있다.

#### 샘플링 속도

샘플링 속도는 센서 값을 1초에 몇 번 읽는지를 의미한다.  
단위는 Hz를 사용한다.

예를 들어 센서 값을 1초에 100번 읽으면 샘플링 속도는 100Hz이다.

```text
100Hz = 1초에 100번 측정
```

샘플링 속도가 너무 낮으면 빠르게 변하는 신호를 놓칠 수 있다.  
반대로 샘플링 속도가 너무 높으면 불필요한 데이터가 많아지고, 마이크로컨트롤러의 처리 부담이 커질 수 있다.

---

### 1.6 센서의 노이즈와 필터링

센서 노이즈란 실제 측정값과 관계없이 센서 출력에 섞이는 불필요한 신호를 말한다.  
노이즈가 발생하면 센서 값이 흔들리거나 잘못된 판단을 할 수 있다.

센서 노이즈의 원인은 다음과 같다.

- 주변 빛의 변화
- 전원 불안정
- 전자기 간섭
- 센서 자체의 오차
- 배선 접촉 불량
- 진동이나 외부 충격

노이즈를 줄이기 위해 필터링을 사용한다.

#### 이동 평균 필터

여러 번 측정한 값을 평균 내어 사용하는 방식이다.  
센서 값이 갑자기 튀는 현상을 줄일 수 있다.

예시:

```text
측정값: 100, 102, 98, 101, 99
평균값: 100
```

#### 임계값 처리

특정 기준값을 정해놓고 그 기준보다 크거나 작을 때만 상태를 판단하는 방식이다.  
라인 감지에서 검은색과 흰색을 구분할 때 자주 사용된다.

예시:

```text
센서값 < 500 → 검은색
센서값 >= 500 → 흰색
```

#### 히스테리시스

ON 기준값과 OFF 기준값을 다르게 설정하여 경계 근처에서 값이 흔들리는 문제를 줄이는 방법이다.  
센서 값이 임계값 근처에서 계속 바뀌는 채터링 현상을 줄일 수 있다.

## 2. IR 센서

### 2.1 IR 센서의 기본 원리

IR 센서는 적외선(Infrared)을 이용하여 물체나 표면의 상태를 감지하는 센서이다.  
적외선은 사람 눈에는 보이지 않지만, 센서를 통해 감지할 수 있는 빛의 한 종류이다.

라인 트레이싱에 사용되는 IR 센서는 보통 적외선 LED에서 빛을 내보내고, 그 빛이 바닥에 반사되어 돌아오는 양을 포토트랜지스터나 포토다이오드가 감지하는 방식으로 동작한다.

표면 색상에 따른 반사 특성은 다음과 같다.

| 표면 색상 | 적외선 반사량 | 센서 반응 |
|---|---|---|
| 흰색 바탕 | 반사량이 많음 | 센서값이 크게 나타남 |
| 검은색 선 | 반사량이 적음 | 센서값이 작게 나타남 |

검은색은 빛을 많이 흡수하고, 흰색은 빛을 많이 반사한다.  
따라서 IR 센서를 사용하면 바닥의 검은색 라인과 흰색 바탕을 구분할 수 있다.

---

### 2.2 IR 센서의 구성 요소

IR 센서는 보통 다음과 같은 구성 요소로 이루어진다.

| 구성 요소 | 역할 |
|---|---|
| IR LED | 적외선을 방출한다 |
| 포토다이오드 또는 포토트랜지스터 | 반사되어 돌아오는 적외선을 감지한다 |
| 비교기 회로 | 센서 값을 기준값과 비교하여 디지털 신호로 변환한다 |
| 가변저항 | 감지 민감도 또는 임계값을 조절한다 |
| 출력 핀 | 아날로그 또는 디지털 신호를 마이크로컨트롤러로 전달한다 |

일부 IR 센서는 아날로그 출력만 제공하고, 일부는 디지털 출력만 제공한다.  
또한 아날로그 출력과 디지털 출력을 모두 제공하는 모듈도 있다.

---

### 2.3 라인 트레이싱을 위한 라인 감지에 적합한 IR 센서의 특성

라인 트레이싱에 적합한 IR 센서는 다음과 같은 특성을 가지는 것이 좋다.

#### 1. 빠른 응답 시간

라인트레이서 로봇은 이동하면서 계속 바닥을 감지해야 한다.  
따라서 센서의 응답 시간이 짧아야 빠른 주행에서도 라인을 놓치지 않는다.

#### 2. 흑백 구분이 명확해야 함

검은색 라인과 흰색 바탕의 반사량 차이를 안정적으로 구분할 수 있어야 한다.  
센서값의 차이가 클수록 제어가 쉬워진다.

#### 3. 적절한 감지 거리

IR 센서는 바닥과 너무 멀면 반사광이 약해지고, 너무 가까우면 감지가 불안정할 수 있다.  
라인 트레이싱에서는 보통 센서와 바닥 사이의 거리를 일정하게 유지하는 것이 중요하다.

#### 4. 외부광에 강해야 함

실내 조명이나 햇빛이 센서에 영향을 줄 수 있다.  
따라서 외부광의 영향을 줄일 수 있는 구조나 필터링이 필요하다.

#### 5. 여러 개의 센서를 사용할 수 있어야 함

라인의 위치를 더 정확히 파악하려면 센서를 여러 개 배열하는 것이 좋다.  
예를 들어 왼쪽, 가운데, 오른쪽에 IR 센서를 배치하면 로봇이 라인의 방향을 판단할 수 있다.

---

### 2.4 IR 센서의 신호 처리

IR 센서의 신호 처리는 센서가 읽은 값을 바탕으로 검은색 라인과 흰색 바탕을 판단하는 과정이다.

#### 아날로그 출력 처리

아날로그 IR 센서는 반사된 적외선의 양에 따라 연속적인 값을 출력한다.  
마이크로컨트롤러에서는 이 값을 읽고 임계값을 기준으로 판단한다.

예시 코드 개념:

```c
int sensorValue = analogRead(A0);

if (sensorValue < 500) {
    // 검은색 라인 감지
} else {
    // 흰색 바탕 감지
}
```

아날로그 방식은 센서값을 세밀하게 확인할 수 있어 정밀 제어에 유리하다.

#### 디지털 출력 처리

디지털 IR 센서는 모듈 내부의 비교기 회로가 기준값을 판단하여 HIGH 또는 LOW를 출력한다.

예시 코드 개념:

```c
int sensorState = digitalRead(2);

if (sensorState == LOW) {
    // 검은색 라인 감지
} else {
    // 흰색 바탕 감지
}
```

디지털 방식은 사용하기 쉽고 코드가 간단하지만, 세밀한 반사량 차이를 알기 어렵다.

#### 여러 센서를 이용한 방향 판단

라인트레이서에서는 여러 개의 IR 센서를 사용하여 로봇이 어느 방향으로 움직여야 하는지 판단한다.

예시:

| 왼쪽 센서 | 가운데 센서 | 오른쪽 센서 | 판단 |
|---|---|---|---|
| 흰색 | 검은색 | 흰색 | 직진 |
| 검은색 | 흰색 | 흰색 | 왼쪽으로 치우침 → 왼쪽 보정 |
| 흰색 | 흰색 | 검은색 | 오른쪽으로 치우침 → 오른쪽 보정 |
| 검은색 | 검은색 | 흰색 | 왼쪽 곡선 |
| 흰색 | 검은색 | 검은색 | 오른쪽 곡선 |

이 판단 결과를 바탕으로 왼쪽 모터와 오른쪽 모터의 속도를 다르게 제어하면 라인을 따라 이동할 수 있다.

---

### 2.5 IR 센서의 배치 시에 유의할 사항

IR 센서를 배치할 때는 다음 사항을 고려해야 한다.

#### 1. 바닥과의 거리 유지

센서와 바닥 사이의 거리가 일정해야 안정적인 값을 얻을 수 있다.  
거리가 너무 멀면 반사광이 약해지고, 너무 가까우면 센서값이 포화되거나 불안정할 수 있다.

#### 2. 센서 간 간격

센서 간격이 너무 좁으면 라인의 위치 변화를 구분하기 어렵고, 너무 넓으면 라인을 놓칠 수 있다.  
라인의 폭에 맞게 센서 간격을 조절해야 한다.

#### 3. 센서의 방향

센서는 바닥을 향해 수직 또는 일정한 각도로 배치해야 한다.  
센서가 기울어지면 반사광이 제대로 돌아오지 않아 측정값이 불안정해질 수 있다.

#### 4. 외부 빛 차단

햇빛이나 강한 조명은 IR 센서의 측정값에 영향을 줄 수 있다.  
필요하다면 센서 주변에 차광 구조를 만들거나 소프트웨어 필터링을 적용해야 한다.

#### 5. 로봇 전면부 배치

라인트레이서에서는 센서를 로봇의 앞쪽에 배치하는 것이 일반적이다.  
센서가 앞쪽에 있으면 로봇이 라인의 변화를 먼저 감지하고 방향을 수정할 수 있다.

#### 6. 센서 높이 보정

로봇이 움직이면서 진동이 발생하면 센서와 바닥 사이의 거리가 변할 수 있다.  
따라서 센서 브래킷을 단단히 고정하고, 주행 중 흔들림을 줄여야 한다.

---

## 3. 결론

센서는 로봇이나 자동화 시스템에서 외부 환경을 감지하고 제어 시스템에 필요한 정보를 제공하는 중요한 장치이다.  
센서의 성능을 이해하기 위해서는 신호 유형, 해상도, 정확도, 응답 시간, 샘플링 속도, 노이즈와 필터링 개념을 알아야 한다.

IR 센서는 적외선을 이용하여 표면의 반사량 차이를 감지하는 센서이다.  
검은색과 흰색의 반사율 차이를 이용할 수 있기 때문에 라인트레이서 로봇의 라인 감지에 많이 사용된다.

라인 트레이싱에서 IR 센서를 안정적으로 사용하려면 센서와 바닥 사이의 거리, 센서 간격, 외부광 영향, 센서 방향, 노이즈 필터링 등을 고려해야 한다.  
이러한 요소를 적절히 조정하면 로봇이 라인을 정확하게 감지하고 안정적으로 주행할 수 있다.
